# RE5 Brand Identity v4

## Brand Architecture

**RE5 is the primary brand.** Bald Research is the publisher credit only — small, italic, footer-position. Never invert this.

**Tagline (primary):** Less noise. More follicles.
**Tagline (secondary):** Stop guessing. Start growing.
**System tagline:** Ancient roots. Modern science. One system.

## Locked Colour Palette

| Name | Hex | Usage |
|---|---|---|
| Lab Paper | `#FAFAF7` | Page background, primary surface |
| Carbon Ink | `#1A1A1A` | Body text, dark cards, primary type |
| Saffron | `#E8893D` | Action — CTAs, highlights, eyebrow tags. Use sparingly: max 5 instances per page. |
| Celadon Teal | `#2F5F6B` | System / framework — phase numbers, system labels, secondary highlights |
| Stone Gray | `#8A8578` | Secondary text, metadata, mono labels |
| Soft | `#5C5852` | Body text on paper (softer than Carbon Ink) |
| Border | `#E0DDD5` | Dividers, card borders, hairlines |

**Critical:** Saffron is the action colour. If something is not an action, a CTA, or critical emphasis, it is NOT saffron. Overuse of saffron is the most common drift error.

**Celadon teal is `#2F5F6B`** — NOT the older `#2F6B57` from earlier brand iterations.

## Typography System

| Font | Use For | Weight |
|---|---|---|
| Nunito Sans | Display headings, body copy, wordmark | 400 / 500 / 600 / 700 / 800 |
| IBM Plex Mono | Eyebrows, data labels, stat numbers, tags | 400 / 500 / 600 |
| Ma Shan Zheng | The 育 seal character ONLY — not for any other text | regular |

**Letter-spacing on headings:** -0.02em (slightly tightened)
**Letter-spacing on eyebrows:** 0.18em (loose, mono caps)
**Eastern script in body copy** (中医, 한방, आयुर्वेद, 인삼, 女贞子, etc.) renders in system sans-serif at color celadon `#2F5F6B`.

## The 育 Seal

The 育 character is the cultural anchor of the brand. It appears in:
- Page footers / cover corners
- Inside saffron rounded-rectangle frames (1.2pt border, ~3mm radius)
- At 50–60% opacity when used as background watermark
- Renders in Ma Shan Zheng font, saffron color `#E8893D`

The seal is NOT used in:
- Body copy
- Logo wordmarks (RE5 is type-only Nunito Sans 800)
- Decoration without purpose

## Logo / Wordmark

**RE5 wordmark** — type-only, Nunito Sans 800, letter-spacing -1pt. The "RE" is carbon ink `#1A1A1A`, the "5" is saffron `#E8893D`. No bracket, no decoration. Just the two-character mark.

## Hard Brand Rules

These are non-negotiable. Violating them is brand drift, regardless of how good the language sounds.

1. **No marketing hype.** Never write "game changer", "revolutionary", "breakthrough", "secret", "doctors hate this", or any influencer-language register.
2. **Honest ceilings.** Every intervention has its limit declared. Never claim guaranteed results.
3. **No fake before-and-afters.** No stock photos posed as buyer results. No comparison images without verifiable provenance.
4. **Natural-first philosophy.** Pharmaceuticals are additions, not anchors. Never dismiss pharma ideologically. Never push pharma commercially.
5. **Mechanism over aesthetics.** Prefer real biological mechanism (Wnt/β-catenin, TGF-β/Smad, NF-κB, eNOS) over sensory or emotional language.
6. **Restraint as default.** "Never say" language (e.g., "game changer") is acceptable when used selectively for contrast — power comes from restraint.
7. **Timelines:** 6-month minimum, 12-month benchmark, 2-year ceiling. Never promise faster than this.

## Locked Ingredient Specifications

These appear across all RE5 content. Do not deviate.

- **Saw palmetto** — always CO₂ extract
- **Zinc** — capped at 15mg
- **Curcumin** — never piperine. Meriva or BCM-95 only. CYP3A4 interaction with finasteride is the reason.
- **Ashwagandha** — strictly conditional. Stress-dominant only. KSM-66 form. Cycle 8–12 weeks on / 4 off.
- **Omega-3** — evening stack only, triglyceride form, IFOS certified
- **Melatonin (topical)** — only inside the Daily Activation Tonic, never standalone
- **Iron** — conditional (ferritin <70). Bisglycinate form. Paired with Amla. Separated from zinc by 6 hours.
- **Pumpkin seed oil** — included in ALL variants (men and women)
- **GHK-Cu** — core regenerative tool. Biphasic dose-response. Post-needling sting test gates use.
- **AHK-Cu** — Official Support, not default. Single-study evidence. Honest ceiling stated.
- **Oils included:** jojoba (base), bhringraj (active), rosemary (capped 5%), black seed, amla
- **Oils excluded:** castor oil, peppermint oil, coconut oil
- **Microneedling depth:** 0.5–0.75mm men, 0.25mm women first two sessions
- **RE5 Hybrid:** dual-route hormonal strategy — finasteride alternating days + botanical bridge. Not simple EOD.
